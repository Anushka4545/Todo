# Anushka Todo


## How to install
Step 1:

    git clone https://github.com/Anushka4545/Todo.git

Step 2:
    Enter into project Directory

Step 3: Open Terminal

Step 4: 

    npm install

step 5:

    npm run dev

Now Application is ready to use
